# 🌤 Smart Weather City Manager

## 📌 About the Project

This is a Flask-based web application that allows users to manage cities and view their real-time weather information using an API.

---

## ✨ Features

* Add new cities
* View weather of cities
* Delete cities
* Shows temperature and weather description
* Temperature category (Hot / Cold / Moderate)

---

## 🛠️ Technologies Used

* Python (Flask)
* SQLite Database
* OpenWeather API
* HTML / CSS

---

## ▶️ How to Run

### 1. Install dependencies

```bash
pip install flask flask_sqlalchemy requests
```

### 2. Run the application

```bash
python app.py
```

### 3. Open in browser

http://127.0.0.1:5000/

---

## 🗄️ Database

This project uses SQLite database (`weather.db`) to store city names.
